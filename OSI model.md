# OSI‑model

| OSI‑laag | Wat de laag doet |
|----------|--------------------------|
| **1. Physical** | Verstuurt bits over de draad of door de lucht (kabel, wifi‑signaal). |
| **2. Data Link** | Regelt communicatie tussen apparaten via MAC‑adressen (Ethernet/Wi‑Fi frames). |
| **3. Network** | Verstuurt pakketten tussen netwerken via IP‑adressen. |
| **4. Transport** | Zorgt voor poorten en verbindingen (TCP/UDP). |
| ***5. Session*** | Beheert sessies tussen twee systemen (aan/uit, synchronisatie). |
| ***6. Presentation*** | Zorgt voor codering, compressie en encryptie (zoals TLS). |
| **7. Application** | Interface voor applicaties: HTTP, DNS, SMTP, FTP, DHCP… |
